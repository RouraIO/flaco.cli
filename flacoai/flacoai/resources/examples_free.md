# Flaco AI Examples (FREE)

These examples work on the FREE tier.

## 1) Review your code (read-only suggestions)
Run a review of your repo and get actionable findings:

- In chat: `/review`

## 2) Ask questions without editing files
Use `/ask` to get explanations and guidance without changing anything:

- `/ask What does this repo do end-to-end?`
- `/ask Where is auth handled?`

## 3) Request code changes
Use `/code` to ask for concrete edits:

- `/code Add input validation to the signup endpoint.`
- `/code Refactor this module to be easier to test.`

## 4) Switch modes quickly
If you run these commands without arguments, you’ll switch modes:

- `/ask`
- `/code`
- `/architect`

## 5) Check your license status
FREE users are “Active” by default:

- `/license`

If you purchased a license:

- `/license activate you@company.com FLACO-XXXXXXXX-XXXXXXXX-XXXXXXXX`

---

Want PRO examples?

- Run: `/examples pro`
- Or: `/license upgrade`
